# 🏺 大唐·时空幻影镜 (The Tang Sancai Holographic Time Mirror)

这是一个将 **多模态 AI 视觉理解 (Gemma 4)** 与 **实时动作捕捉 (MediaPipe)** 深度结合的数字化非遗互动装置原型。参与者可以通过自己的今日穿搭（OOTD）唤醒相匹配的大唐三彩乐俑，并通过肢体动作（合十、起舞、旋转）引导乐俑完成三彩釉色、青花瓷、黄金金釉等跨越千年的釉色华美变幻。

---

## ✨ 核心亮点 (Key Features)

- 🤖 **Gemma 4 本地多模态穿搭感应**：利用本地运行的 Gemma 4 视觉模型自动捕获参与者着装，智能识别色系与穿搭风格，从大唐文物库中匹配特质最契合的唐俑角色。
- 🎥 **双视频缓冲区无缝切换**：采用双 `<video>` 播放器底层交替缓冲渲染，通过 `300ms` CSS 交叉淡入淡出，消除视频切换间的黑屏闪烁，实现电影级平滑转场。
- 🌊 **EMA 指数移动平均滤波**：在 MediaPipe Pose 检测之上，引入指数衰减滤波平滑算法，过滤网络摄像头带来的像素噪声抖动，提供极高精准度与响应速度的动作手势触发。
- 🌀 **肢体交互状态机**：
  - **双手合十**：触发开幕 Reveal 效果。
  - **模仿姿势**：姿势相似度匹配，唤醒舞蹈。
  - **360°旋转**：检测转圈动作，动态触发唐三彩服饰的青花瓷、黄金金釉材质切换。
- 📜 **个性化拍立得贴纸与 AI 祝词**：由 Gemma 4 根据穿搭色系，结合语义分析实时生成精美的四字国风古典祝词（如“绯红霓裳”、“金玉满堂”），生成一张专属的拍立得纪念海报。

## 🚀 快速开始 (Quick Start)

这是一个静态前端项目，配合本地运行的 Ollama (Gemma 4) 服务即可直接启动：

### 1. 运行本地网页服务 (Local Hosting)
打开终端进入项目根目录，运行本地 HTTP 静态服务器：

```bash
# 使用 Python3 快速建立本地服务器
python3 -m http.server 8000
```
启动后在浏览器中访问：**`http://localhost:8000`**

### 2. 部署本地 Gemma 4 视觉大模型
项目采用本地免安装的单文件版 Ollama 进行部署：

1. **下载与准备**：
   项目根目录下已包含预编译的可执行文件 `./ollama`。如果不存在，可运行以下命令下载（免 Sudo 权限）：
   ```bash
   curl -L https://ollama.com/download/ollama-linux-amd64.tar.zst -o ollama.tar.zst
   # 解压出 bin/ollama 并创建软链接
   mkdir -p ollama_dist && unzstd -c ollama.tar.zst | tar -xf - -C ./ollama_dist
   ln -sf ./ollama_dist/bin/ollama ./ollama
   ```

2. **启动 Ollama 接口服务**（注意：必须显式允许跨域）：
   ```bash
   OLLAMA_ORIGINS="*" OLLAMA_HOST="0.0.0.0" ./ollama serve
   ```

3. **拉取 Gemma 4 视觉模型**：
   新开一个终端窗口，在项目根目录下运行：
   ```bash
   ./ollama pull gemma4:e2b
   ```

4. **网页端使用**：
   刷新本地网页后，系统默认将直接向本地的 `http://localhost:11434` 发起 Gemma 4 穿搭识别请求。无需额外配置！

> [!NOTE]
> Gemma 4 运行在本地 CPU 环境下时，单次推理时间大约在 15-25 秒左右（取决于处理器性能）。为此，前端的接口请求超时时间已针对性延长至 45 秒，以确保本地 CPU 推理能够稳定返回结果而不被浏览器提前掐断。
