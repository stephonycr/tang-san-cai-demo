# 🏺 大唐·时空幻影镜 (The Tang Sancai Holographic Time Mirror)

这是一个将 **多模态 AI 视觉理解 (Gemma/Gemini)** 与 **实时动作捕捉 (MediaPipe)** 深度结合的数字化非遗互动装置原型。参与者可以通过自己的今日穿搭（OOTD）唤醒相匹配的大唐三彩乐俑，并通过肢体动作（合十、起舞、旋转）引导乐俑完成三彩釉色、青花瓷、黄金金釉等跨越千年的釉色华美变幻。

---

## ✨ 核心亮点 (Key Features)

- 🤖 **Gemma/Gemini 多模态穿搭感应**：自动捕获参与者着装，智能识别色系与穿搭风格，从大唐文物库中匹配特质最契合的唐俑角色。
- 🎥 **双视频缓冲区无缝切换**：采用双 `<video>` 播放器底层交替缓冲渲染，通过 `300ms` CSS 交叉淡入淡出，消除视频切换间的黑屏闪烁，实现电影级平滑转场。
- 🌊 **EMA 指数移动平均滤波**：在 MediaPipe Pose 检测之上，引入指数衰减滤波平滑算法，过滤网络摄像头带来的像素噪声抖动，提供极高精准度与响应速度的动作手势触发。
- 🌀 **肢体交互状态机**：
  - **双手合十**：触发开幕 Reveal 效果。
  - **模仿姿势**：姿势相似度匹配，唤醒舞蹈。
  - **360°旋转**：检测转圈动作，动态触发唐三彩服饰的青花瓷、黄金金釉材质切换。
- 📜 **个性化拍立得贴纸与 AI 祝词**：自动根据穿搭色系，结合大模型语义分析生成精美的四字国风古典祝词（如“绯红霓裳”、“金玉满堂”），生成一张专属的拍立得纪念海报。

## 🚀 快速开始 (Quick Start)

这是一个**纯前端静态项目**，无需复杂的后端编译即可直接启动：

### 1. 本地运行 (Local Hosting)
打开终端进入项目根目录，运行本地 HTTP 静态服务器：

```bash
# 使用 Python3 快速建立本地服务器
python3 -m http.server 8000
```
启动后在浏览器中访问：**`http://localhost:8000`**

### 2. 线上发布 (GitHub Pages)
您可以直接将其推送到 GitHub 并开启 **GitHub Pages** 服务：
* 根目录下已包含 `.nojekyll` 配置文件，用于告诉 GitHub 直接发布静态页面，绕过 Jekyll 编译限制，从而保证大体积互动视频资源快速加载。

## ⚙️ AI 穿搭与祝福语功能配置

项目支持两种多模态大模型推理路线：

### 路线 A：云端 Gemini API (推荐，最便捷)
1. 在项目网页的配置面板中，粘贴您的 **Gemini API Key**。
2. 保持 **"Use Local VLM (Gemma)"** 复选框为**未勾选状态**。
3. 系统将直接通过浏览器前端安全地调用云端大模型完成穿搭和祝福语分析。

### 路线 B：本地 Gemma / Moondream (适合展会无网部署)
1. 在本地机器安装 [Ollama](https://ollama.com/) 并拉取轻量级 VLM 模型（例如 `moondream`）：
   ```bash
   ollama pull moondream
   ```
2. 开启跨域允许启动 Ollama 服务（注意：必须加上 `OLLAMA_ORIGINS` 环境变量以避开浏览器跨域安全限制）：
   ```bash
   OLLAMA_ORIGINS="*" OLLAMA_HOST="0.0.0.0" ollama serve
   ```
3. 在项目网页配置面板中，勾选 **"Use Local VLM (Gemma)"** 并确认 Endpoint 为本地接口地址。
